==============================
Account Reconciliation – Odoo 18
==============================

.. image:: banner.png
   :alt: Account Reconciliation
   :align: center


Account Reconciliation is an Odoo 19 module that simplifies the reconciliation process
in Odoo Community. It allows users to easily reconcile multiple journals by grouping them
based on partners, reducing manual effort and improving accounting accuracy.

With an intuitive interface and guided reconciliation wizard, users can quickly select
the appropriate account and reconcile pending entries, ensuring clean and well-maintained
financial records.
